CHATTUP

A web-based chat and social media application, built using Django, DjangoChannels, Websocket API/protocol, Redis and channel layers(for scope visibility of connected users. Using channel layers, I was able to network all connected clients into what we call a "room", which permits distributed messages).

ECOMMERCE

An Ecommerce application for a home appliance niche-- beds for babys. Anyone can purchase products anonymously, without having to signup in the app. And for signing up, one could sign up as a vendor or a buyer. All vendors have the priviledge of uploading their products to the app, editing them, and deleting them at will. However, there are default products available in the app, as well.

REST API
